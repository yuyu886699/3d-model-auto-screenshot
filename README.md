# 3d-model-auto-screenshot# 3D 模型多视角自动截图工具

一个基于 Web 的 3D 模型多视角自动截图 Demo。

## 功能特点

- 支持上传 GLB / GLTF 格式模型
- 支持前、后、左、右、顶、底、等轴测等多视角截图
- 支持自定义截图视角
- 支持透明、深色、浅色背景
- 支持 512 / 1024 / 2048 分辨率输出
- 支持截图结果预览
- 支持一键打包下载 ZIP

## 使用方式

1. 下载本项目
2. 双击打开 `index.html`
3. 上传本地 `.glb` 或 `.gltf` 模型
4. 点击“立即自动多视角截图”
5. 下载单张图片或 ZIP 压缩包

## 说明

这是一个设计 Demo，用于验证“三维模型自动多角度截图”的工作流。  
它适合用于需求沟通、方案演示和轻量化工具验证，不代表完整生产级软件。

## 注意事项

- 请勿上传涉密、敏感或未经授权的模型文件
- 当前版本更适合单文件 `.glb` 模型
- 大体积模型可能会影响浏览器性能
- 本工具在本地浏览器中运行，不主动上传模型数据到服务器
